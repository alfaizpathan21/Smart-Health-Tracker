# 💙 Smart Health Tracker

Smart Health Tracker is a fully functional PHP + MySQL web application designed to help users monitor and improve their daily wellness. It allows users to track steps, water intake, sleep hours, BMI, and personal fitness goals through an interactive dashboard with a modern glassmorphism UI.

## 🚀 Features

* User Registration & Login Authentication
* Personal Dashboard with Health Metrics
* Daily Step Tracking
* Water Intake Monitoring
* Sleep Hours Tracking
* BMI Calculator with Live Preview
* Custom Daily Goals
* 7-Day Health History Records
* AJAX-Based Real-Time Updates
* Fully Responsive Design

## 🛠️ Tech Stack

* Frontend: HTML5, CSS3, JavaScript
* Backend: PHP
* Database: MySQL
* Server: XAMPP / Apache

## ⚙️ Installation Guide

1. Clone Repository
   git clone https://github.com/your-username/health_tracker.git

2. Move Project Folder to:
   C:\xampp\htdocs\

3. Import Database

* Open phpMyAdmin
* Create database: health_tracker
* Import file: health_tracker.sql

4. Configure Database in config.php

define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_NAME', 'health_tracker');

5. Run Project in Browser
   http://localhost/health_tracker/homepage.php
